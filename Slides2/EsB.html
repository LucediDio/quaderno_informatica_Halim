Accesso a pagina riservata



Spiegazione del Codice

Questo codice HTML e PHP serve per creare una semplice pagina web che gestisce l'accesso a una sezione riservata. In pratica, la pagina contiene un form che richiede all'utente di inserire un nome utente e una password. Quando l'utente invia il form, i dati vengono inviati al server, dove vengono elaborati da un codice PHP.

Nel form HTML, vediamo due campi di input:

  • Username: un campo di testo dove l'utente deve inserire il proprio nome utente.
  • Password: un campo di password dove l'utente deve inserire la propria password. Il testo della password Ã¨ nascosto.

Quando l'utente preme il pulsante Invia, i dati vengono inviati al file EsB1.php tramite il metodo POST.

Spiegazione del Codice PHP

All'interno del file EsB1.php, il codice PHP controlla se il nome utente e la password sono corretti. In particolare:

  • Il nome utente corretto e' admin.
  • La password corretta e' passwd.

Se i dati inseriti dall'utente non corrispondono a questi valori, viene mostrato un messaggio che dice "Nome utente o password sbagliati. Accesso negato!".

Se i dati sono corretti,viene mostrato un messaggio di benvenuto che include il nome utente inserito.

Se l'utente non inserisce dati, tramite il metodo isset controllo se le variabili siano == a null e in tal caso gli dico di riprovare.
